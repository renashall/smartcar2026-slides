# Organization Notes

General coaching and administrative reference for AI Code Academy classes.

## Company

- **Name:** AI Code Academy
- **Site:** [aicodeacademy.com](https://aicodeacademy.com)
- **Lessons hosted via:** [Outschool](https://outschool.com) or [Lessonbook.ai](https://lessonbook.ai)
- **Primary communication:** Discord (coaches, admins, etc.)

## Final Meeting Announcement

- **Template:** [Final Meeting Announcement (Notion)](https://aicode-coach-resources.notion.site/Final-Meeting-Announcement-9ff5b8cc4d9b4d8ba2691eb52f1b023e)
- Copy and paste onto the class page, editing or adding content as necessary.

## Certificates

- **Generator:** [scg.aicode101.com](https://scg.aicode101.com/)
- Used to generate certificates after final project presentations (when a course includes them).
- Issue class certificates to **all** students.
- Choose the **top three winners** based on final project presentation quality:
  - One **High Honors** student
  - Two **Honors** students
  - **Regular** certificates for all other students
- Announce the winners:
  - Post a winners message to the class page.
  - Post in the `final-project-winners` Discord channel with your class name, meeting time, and winners' names.
- Full instructions are available on the certificate site.

## Coach Checklist

- Always post a class announcement.
- Always post in the proper Discord channel for absent students.
- Always start a new class with an announcement introducing yourself.
