Rounded white surface panel — the base container for lessons, material tiles and slide blocks.

```jsx
<Card padding="md" elevation="sm">Lesson content…</Card>
<Card interactive accent="teal">Hoverable tile with teal top bar</Card>
```

Props: `padding` (sm/md/lg), `elevation` (none/xs/sm/md/lg), `interactive` (lifts on hover), `accent` (navy/teal/amber top bar). Composes anything — pair with Badge, Button, CodeBlock.
