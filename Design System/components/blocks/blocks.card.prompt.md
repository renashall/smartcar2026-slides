Slide block components — the reusable panels that appear on lesson and coach slides. InfoCard (filled navy "Questions?" / "Up next" call-out), IconDescCard (icon + title + description box, the thank-you "ways" pattern, with a `glass` variant for dark slides), TierList (certificate tiers), and MiniChecklist (the coach "every class" reminder list). Slides render these from the bundle so styling stays in sync.
