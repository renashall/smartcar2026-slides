Tier list — count tag + bold label rows. The certificate-tiers pattern, for a dark card.

```jsx
<TierList items={[
  { tag: "×1", label: "High Honors" },
  { tag: "×2", label: "Honors" },
  { tag: "all", label: "Regular — everyone else" },
]} />
```

Props: `items` of `{ tag, label }`. Amber tags with white text, white bold labels.
