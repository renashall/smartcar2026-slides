Mini checklist — compact ✓ items. The coach "every class" reminder pattern; `onDark` (default) for navy cards.

```jsx
<MiniChecklist items={[
  "Post a class announcement",
  "Introductions if needed",
  "Note absences in #attendance",
]} />
```

Props: `items` (strings), `onDark` (white text), `divided` (hairlines). Teal ticks on dark, green on light.
