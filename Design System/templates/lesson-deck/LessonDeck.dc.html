Pick a self-driving mode

Open the lesson file and change just the settings at the top — no other edits needed.

1

Set DEMO_MODE to choose how the car thinks.

2

"sonic" steers around obstacles with the ultrasonic sensor.

3

Tune DRIVE_SPEED — start gentle, then go faster.

lesson_4_autonomous_modes.py 
# pick one self-driving mode
DEMO_MODE   = "sonic"   # or "light"
DRIVE_SPEED = 1200
TURN_SPEED  = 1500

# closer than this (cm) = obstacle
BLOCKED     = 30

Common issues

If your car isn't behaving, it's almost always one of these.

Permission denied / mmap() failed
The LED lessons need root — run them with sudo.
Can't connect to the Pi
Check both devices are on the same Wi-Fi and the IP is correct.
Car drives but won't steer
The servo cable is in the wrong header — match channel 0 to pan.
?
Questions?

If anything isn't working, ask now — we'll fix it together before we move on.
