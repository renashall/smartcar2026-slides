# Coach reference — running an AI Code Academy class

Operational notes for coaches using this design system. Student-facing slide
templates that match these workflows live in `slides/` (`thanks.html`,
`course-wrapup.html`, `common-issues.html`).

## Communication
- **Discord is the primary communication channel** for coaches, admins and
  students — help channels, announcements, project showcases.
- **Always post a class announcement** before/after each session on the class page.
- **Always post in the proper Discord channel for absent students** so they can
  catch up.
- **Start every new class with an announcement introducing yourself.**

## End-of-course: final announcement
- **Final announcement template:**
  https://aicode-coach-resources.notion.site/Final-Meeting-Announcement-9ff5b8cc4d9b4d8ba2691eb52f1b023e
  Copy and paste onto the class page. Edit / add content as necessary.
  (The Notion page renders client-side and couldn't be inlined here — open the
  link directly.)

## Certificates (courses with a final project)
- **Certificate generator:** https://scg.aicode101.com/ — full instructions are
  on the site.
- Issued after final project presentations. **Every student gets a class
  certificate.**
- Choose the **top three** based on final-project presentation quality:
  - **1 × High Honors**
  - **2 × Honors**
  - Regular certificates to all other students.
- Post a class-page announcement naming the winners.
- Post your **class name, meeting time and winners' names** in the
  **`#final-project-winners`** Discord channel.
