Build the file · chunk 3 / 8

face_center() & center_distance()

The middle of a face box, and the straight-line distance between two faces.

face_center → the middle point of a face box.

center_distance → how far apart two faces are.

It's the Pythagorean theorem — a²+b²=c². Hello, math class.

lesson_8_multiple_face_detection.py
def face_center(face_box):
    """Return the middle point of a face box."""
    x, y, w, h = face_box
    return int(x + w / 2), int(y + h / 2)


def center_distance(first_face, second_face):
    """Return the distance between two face centers."""
    first_x, first_y = face_center(first_face)
    second_x, second_y = face_center(second_face)
    return ((first_x - second_x) ** 2
            + (first_y - second_y) ** 2) ** 0.5

Build the file · chunk 4 / 8

choose_target(): follow the same person

Returns two things: the face and a label we show on screen.

No target yet → start with the biggest face.

Have a target → pick the closest face to last frame.

Too far from any face → restart with the biggest.

lesson_8_multiple_face_detection.py
def choose_target(faces, previous_target):
    """Choose one face to follow, reusing the previous target."""
    if len(faces) == 0:
        return None, "none"
    if previous_target is None:
        return face.biggest_face(faces), "largest"

    closest_face = min(faces,
        key=lambda f: center_distance(f, previous_target))
    if center_distance(closest_face, previous_target) <= LOCK_DISTANCE:
        return closest_face, "locked"
    return face.biggest_face(faces), "largest"

draw_faces(): mark all, highlight the target

The takeaway: boxes for all, a circle for the target.

Green box around every face.

A yellow circle + label on the one we follow.

Don't dwell on the drawing args.

lesson_8_multiple_face_detection.py
def draw_faces(frame, faces, target, target_mode):
    """Draw every face and highlight the target face."""
    for face_box in faces:
        x, y, w, h = face_box
        cv2.rectangle(frame, (x, y), (x + w, y + h), BOX_COLOR, 2)

    if target is not None:
        x, y, w, h = target
        center = face_center(target)
        cv2.circle(frame, center, int((w + h) / 4), TARGET_COLOR, 2)
        cv2.putText(frame, "target: " + target_mode,
                    (x, min(frame.shape[0] - 10, y + h + 18)),
                    cv2.FONT_HERSHEY_SIMPLEX, 0.5, TARGET_COLOR, 1, cv2.LINE_AA)

def draw_detection_summary(frame, faces, target, target_mode):
    """Show a short summary for this video frame."""
    if target is None:
        summary = "faces: 0  target: none"
    else:
        summary = ("faces: " + str(len(faces))
                   + "  target: " + target_mode)
    cv2.putText(frame, summary, (10, 22),
                cv2.FONT_HERSHEY_SIMPLEX, 0.6, TARGET_COLOR, 2, cv2.LINE_AA)

Build the file · chunk 7 / 8

loop(): reuse + lock + draw

Every face.xxx() call is a Lesson 7 function doing the heavy lifting.

Reuse L7 for the camera, detection, head movement.

New: keep a previous_target so the lock carries over.

Press q to quit.

lesson_8_multiple_face_detection.py
def loop():
    """Mark all faces, keep a target lock, show the video."""
    previous_target = None
    while True:
        frame = face.read_frame()           # from Lesson 7
        if frame is None:
            break
        faces = face.detect_faces(frame)    # L7: find EVERY face
        target, target_mode = choose_target(faces, previous_target)
        previous_target = target
        draw_faces(frame, faces, target, target_mode)
        draw_detection_summary(frame, faces, target, target_mode)
        face.track_face(target)             # L7: turn toward target
        cv2.imshow(WINDOW_NAME, frame)
        if cv2.waitKey(1) & 0xFF == ord("q"):
            break
