Sense → Think → Act

Every autonomous robot repeats three steps, fast — many times a second.

1
Sense

Read the sensors — light voltages, or distance ahead.

2
Think

Decide what to do with those numbers — go, turn, or stop.

3
Act

Drive the motors based on the decision.

Our loop() repeats these three steps over and over — that's the whole lesson.

setup(): build only what you need

Both modes drive, so always make the motor — then create only the chosen mode's sensors.

Parts start as None; build them in setup().

The if/else only builds what the mode needs — the rest stay None.

Sonic mode starts the head looking straight ahead.

lesson_4_autonomous_modes.py
motor = None
adc = None
servo = None
ultrasonic = None


def setup():
    """Build just the hardware this mode needs."""
    global motor, adc, servo, ultrasonic
    motor = Motor()                  # both modes drive
    if DEMO_MODE == "light":
        adc = Adc()                  # light sensors
    else:
        servo = Servo()              # head + sensor
        ultrasonic = Ultrasonic()
        servo.setServoPwm("0", HEAD_CENTER)

Lesson 4 — Autonomous modes · the file, chunk 6

light_step(): steer toward light

One pass of Sense → Think → Act for light mode: read both sensors, compare, drive.

Sense: read the left and right light channels.

Think: both bright → straight; one side brighter → turn.

Act: the turn tuples are the same trick as Lesson 2.

lesson_4_autonomous_modes.py
def light_step():
    left = adc.recvADC(LEFT_LIGHT)
    right = adc.recvADC(RIGHT_LIGHT)
    # smaller voltage = brighter
    if left < BRIGHT_ENOUGH and right < BRIGHT_ENOUGH:
        motor.setMotorModel(DRIVE_SPEED, DRIVE_SPEED, DRIVE_SPEED, DRIVE_SPEED)
    elif left - right > SIDE_DIFFERENCE:
        motor.setMotorModel(-TURN_SPEED, -TURN_SPEED, TURN_SPEED, TURN_SPEED)
    elif right - left > SIDE_DIFFERENCE:
        motor.setMotorModel(TURN_SPEED, TURN_SPEED, -TURN_SPEED, -TURN_SPEED)
    else:
        motor.setMotorModel(0, 0, 0, 0)

look(): aim and measure

A small helper — point the head at an angle, wait for it to arrive, then measure.

Turn the head to angle, then read the distance there.

The sleep(0.3) gives the head time to finish moving.

Reused three times in sonic mode — ahead, left, right.

lesson_4_autonomous_modes.py
def look(angle):
    """Aim the head, then measure distance (cm)."""
    servo.setServoPwm("0", angle)
    time.sleep(0.3)                  # let the head settle
    return ultrasonic.get_distance()

sonic_step(): look, then turn

If the path ahead is clear, drive. If blocked, stop, look both ways, and turn to open space.

Clear ahead → drive forward, then return early.

Blocked → stop, measure left and right.

Turn toward the side with more room.

lesson_4_autonomous_modes.py
def sonic_step():
    ahead = look(HEAD_CENTER)
    if ahead > BLOCKED:
        motor.setMotorModel(DRIVE_SPEED, DRIVE_SPEED, DRIVE_SPEED, DRIVE_SPEED)
        return                              # clear road

    motor.setMotorModel(0, 0, 0, 0)         # stop and look
    left = look(HEAD_LEFT)
    right = look(HEAD_RIGHT)
    servo.setServoPwm("0", HEAD_CENTER)    # re-center the head
    if left > right:
        motor.setMotorModel(-TURN_SPEED, -TURN_SPEED, TURN_SPEED, TURN_SPEED)
    else:
        motor.setMotorModel(TURN_SPEED, TURN_SPEED, -TURN_SPEED, -TURN_SPEED)
    time.sleep(0.5)                      # turn briefly

loop(): run the chosen mode

Repeat the right single step forever, with a tiny pause — the Sense → Think → Act loop made real.

DEMO_MODE decides which single step runs each pass.

The while True repeats it many times a second.

A tiny sleep(0.05) keeps the loop from being frantic.

lesson_4_autonomous_modes.py
def loop():
    """Repeat the chosen mode's step over and over."""
    while True:
        if DEMO_MODE == "light":
            light_step()
        else:
            sonic_step()
        time.sleep(0.05)             # a tiny pause

Clean up: destroy()

Stop driving, re-center the head, and close the sensor connection.

setMotorModel(0, 0, 0, 0) stops the wheels.

Re-center the head and i2cClose() the ADC.

Guard with is not None — unused parts were never created.

lesson_4_autonomous_modes.py
def destroy():
    """Stop the car and tidy up the hardware."""
    if motor is not None:
        motor.setMotorModel(0, 0, 0, 0)       # stop
    if servo is not None:
        servo.setServoPwm("0", HEAD_CENTER)   # face front
    if adc is not None:
        adc.i2cClose()                        # close ADC

The main block

The familiar wrapper: try runs the work, finally always cleans up.

try runs setup() then loop().

except KeyboardInterrupt catches Ctrl+C.

finally guarantees destroy() runs no matter what.

Fourth time seeing this block — you should predict it now.

lesson_4_autonomous_modes.py
if __name__ == "__main__":
    try:
        setup()
        loop()
    except KeyboardInterrupt:
        pass
    finally:
        destroy()

Common issues

If the driving looks off, it's almost always one of these.

Turns the wrong way (light)
Remember lower voltage = brighter; check the left/right wiring.
Spins in circles (sonic)
BLOCKED too high, or the head isn't centered. Lower the speeds.
Won't move
Battery low, or DEMO_MODE needs sensors that aren't connected.
Head doesn't turn
Servo is on channel "0"; give it time — look() waits 0.3 s.
Check these first

Most "weird driving" today is a low battery or too-high speed. Rule those out before anything else.
