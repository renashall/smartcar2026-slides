Face position → head movement

setup(): load the detector & connect

Load the face cascade (and check it loaded), then open both connections.

empty() is True if the file failed to load.

Open the video connection (8000)…

…and the command connection (5000). Two connect() calls.

lesson_7_face_tracking.py
def setup():
    global cmd, face_cascade, video_socket, video_file, command_socket
    if len(sys.argv) < 2:
        raise SystemExit("Pass the Pi's IP, e.g. python this.py 192.168.1.50")

    cmd = COMMAND()
    face_cascade = cv2.CascadeClassifier(str(CASCADE_PATH))
    if face_cascade.empty():
        raise FileNotFoundError("Could not load the face file: " + str(CASCADE_PATH))

    pi_ip = sys.argv[1]
    video_socket = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    video_socket.connect((pi_ip, VIDEO_PORT))    # pictures in
    video_file = video_socket.makefile("rb")

    command_socket = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    command_socket.connect((pi_ip, COMMAND_PORT))  # commands out

Build the file · chunk 5 / 12

read_frame(): the same trick as Lesson 6

Read the 4 size-bytes, then the JPEG, then decode it — copied straight from Lesson 6.

Read the 4 size-bytes; turn them into a number.

If the size is 0 → the stream ended.

Reusing a known idea — recognizing it is a real skill.

lesson_7_face_tracking.py
def read_frame():
    """Read one picture, or None when the stream ends."""
    header = video_file.read(4)
    if len(header) != 4:
        return None
    length = struct.unpack("<L", header)[0]
    if length == 0:
        return None
    jpeg = video_file.read(length)
    if len(jpeg) != length:
        return None
    return cv2.imdecode(np.frombuffer(jpeg, np.uint8), cv2.IMREAD_COLOR)

Build the file · chunks 6–7 / 12

detect_faces() & biggest_face()

Grey the picture and find every face — then pick the largest box (the closest person).

detectMultiScale returns a box per face.

Biggest box ≈ closest face (w × h is its area).

Lesson 8 will improve on this assumption.

lesson_7_face_tracking.py
def detect_faces(frame):
    """Return a list of (x, y, w, h) boxes, one per face."""
    gray = cv2.cvtColor(frame, cv2.COLOR_BGR2GRAY)
    # 1.3 and 5 control how carefully it searches.
    return face_cascade.detectMultiScale(gray, 1.3, 5)


def biggest_face(faces):
    """Return the largest face's box, or None if there are none."""
    if len(faces) == 0:
        return None
    return max(faces, key=lambda face: face[2] * face[3])

send_servo(): build a CMD_SERVO command

Reuse the command idea from Lesson 2 — now sent over port 5000.

Format: CMD_SERVO#<channel>#<angle> + a newline.

Text must be sent as bytes — .encode("utf-8").

The same #-joined strings from Lesson 2.

lesson_7_face_tracking.py
def send_servo(channel, angle):
    """Send one command, e.g. CMD_SERVO#0#95."""
    message = (cmd.CMD_SERVO + "#" + channel
               + "#" + str(int(angle)) + "\n")
    command_socket.send(message.encode("utf-8"))

track_face() & clamp(): nudge the head

Turn position into an offset, skip tiny offsets, then nudge pan / tilt within limits.

Dead zone: already centered → don't move.

Nudge pan / tilt toward the face.

clamp() keeps angles inside the servo limits.

lesson_7_face_tracking.py
def track_face(face):
    global pan_angle, tilt_angle
    if face is None:
        return
    x, y, w, h = face
    center_x = x + w / 2
    center_y = y + h / 2
    # offset: -1 at left/top, +1 at right/bottom, 0 in the middle.
    offset_x = (center_x / FRAME_WIDTH - 0.5) * 2
    offset_y = (center_y / FRAME_HEIGHT - 0.5) * 2
    if abs(offset_x) < DEAD_ZONE and abs(offset_y) < DEAD_ZONE:
        return                       # already centered
    pan_angle = clamp(pan_angle + SERVO_STEP * offset_x, PAN_MIN, PAN_MAX)
    tilt_angle = clamp(tilt_angle - SERVO_STEP * offset_y, TILT_MIN, TILT_MAX)
    send_servo(PAN_SERVO, pan_angle)
    send_servo(TILT_SERVO, tilt_angle)


def clamp(value, low, high):
    return max(low, min(high, value))

loop(): find, draw, follow

lesson_7_face_tracking.py
def loop():
    """Read pictures, find the face, follow it, show the video."""
    while True:
        frame = read_frame()
        if frame is None:
            break
        face = biggest_face(detect_faces(frame))
        if face is not None:
            x, y, w, h = face
            cv2.rectangle(frame, (x, y), (x + w, y + h), (0, 255, 0), 2)
            track_face(face)
        cv2.imshow(WINDOW_NAME, frame)
        if cv2.waitKey(1) & 0xFF == ord("q"):
            break

def destroy():
    """Close both connections and the window."""
    if video_file is not None:
        video_file.close()
    if video_socket is not None:
        video_socket.close()
    if command_socket is not None:
        command_socket.close()
    cv2.destroyAllWindows()


if __name__ == "__main__":
    try:
        setup(); loop()
    except KeyboardInterrupt:
        pass
    finally:
        destroy()

Lesson 7 — Face tracking

Common issues

If tracking misbehaves, it's almost always one of these.

Could not load the face file
The cascade path is wrong — confirm it's in Code/Client/.
No green box
More light on your face; face the camera straight on.
Head jitters / overshoots
Raise DEAD_ZONE or lower SERVO_STEP.
Connection refused
Start sudo python3 main.py on the Pi first; check the IP.
Lighting wins

Lighting is the biggest factor in detection quality. Aim a lamp at your face if the box won't appear.
