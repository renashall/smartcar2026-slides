Battery safety

Treat lithium batteries with respect — read this carefully.

Use the correct type for the car (see About_Battery.pdf).

Insert with the right polarity (+ / −), and never short the terminals — keep cells away from coins, keys, metal.

Don't use damaged cells — dented, leaking, or wrapper torn.

Charge with the proper charger; don't leave it charging unattended.

Keep away from heat and water; store in a cool, dry place.

When in doubt

Set a questionable cell aside and don't use it.

Which Raspberry Pi? — Pi 4 or 5

The course runs on a Pi 3B+, 4, or 5 — the Pi 4 is the one we recommend.

Common issues

If something's off, it's almost always one of these.

Won't boot / lightning icon → wrong or weak power adapter (slide 16).

Can't find the Pi → same Wi-Fi? Try the IP instead of .local.

SSH refused / times out → SSH wasn't enabled in Imager; re-flash.

VNC won't connect → not enabled (slide 27); black screen → set a resolution.

git: command not found → install git (slide 32) or use the ZIP.

python not found (Windows) → use py, or reinstall with "Add to PATH".

Venv won't activate → Set-ExecutionPolicy -Scope CurrentUser RemoteSigned.

ModuleNotFoundError → activate .venv, then pip install -r requirements.txt.

Setup script error (Pi) → re-run it, read the last error line, and check the Pi's internet.
